# TelegramGygax

Hopefully some day it'll be a decent text adventure.

## Usage
### Run locally
`bundle exec exe/cli`

### Run Telegram bot
```bash
TELEGRAM_GYGAX_TOKEN=<<your token>> bundle exec exe/bot
```
